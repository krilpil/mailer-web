# Project Brief

## Goals
- Предоставьте веб-приложению Mailfinch Core (Next.js App Router) аутентификацию и основные рабочие процессы электронной почты/домена, присутствующие в репозитории: домены, почтовые ящики, рассылки, шаблоны и потоки подписчиков.
- Интеграция пользовательского интерфейса с маршрутами API сервера и API внешнего почтового провайдера (API администрирования BillionMail и API отправки).
- Создание унифицированного веб-интерфейса для управления доменом, подготовки почтовых ящиков, потоков, связанных с DNS, и сценариев отправки электронной почты.

## Non-Goals
- Не является почтовым провайдером или SMTP-сервисом; интегрируется с внешним провайдером (BillionMail API).
- В этом корневом каталоге репозитория нет определений CI/CD или инфраструктуры.
- Нет замены функциональности BillionMail core на уровне почтового сервера; проект фокусируется на пользовательском интерфейсе продукта, оркестровке и интеграции API.

## Constraints
- Следуйте уровням FSD и общедоступному API через "index.ts".
- Компоненты пользовательского интерфейса не вызывают API напрямую; обратные вызовы передаются с более высоких уровней.
- Используйте скрипты yarn для сборки / lint/ тестирования (соглашение о репозитории).
- Сохраняйте интеграцию с BillionMail на стороне сервера; не предоставляйте доступ к администратору и не отправляйте ключи API клиенту.
- Соблюдайте существующую архитектуру проекта и избегайте нарушения текущих контрактов в обработчиках маршрутов и функциональных модулях.

## Stakeholders
- Команда разработки и сопровождения Mailfinch Core.
- Пользователи веб-панели.
- Администраторы и операторы, управляющие доменами, почтовыми ящиками и рассылками.

## Success Criteria
- Публичные маршруты авторизации и приватные страницы работают и защищены.
- Доменные операции и операции рассылки доступны через UI и API.
- Создание домена, управление почтовыми ящиками и отправка писем через BillionMail работают стабильно.
- Документация и Memory Bank актуальны.
- Ключевые сценарии пользователя проходят без ручной работы в панели BillionMail.
